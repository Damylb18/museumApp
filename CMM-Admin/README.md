# CMM Admin

This part of the project is a .NET based web app and api, making use of an SQLite database. It stores information on museum artificats that will be accessed by the accompanying app. No personal information is stored.
The goal of the part of the project is to make the contents of the app easily modifiable by CMM staff via a user friendly admin panel.

## Deploying
Microsoft recommends publishing .NET web apps with Microsoft Azure, but there are other options available if desired. To publish to Azure, follow the .NET publishing documentation.
https://learn.microsoft.com/en-us/visualstudio/deployment/quickstart-deploy-aspnet-web-app

In order to publish to another type of server, follow their instructions.
